# List All Available Color Theme Names for Data Dictionary Creation.

List All Available Color Theme Names for Data Dictionary Creation.

## Usage

``` r
ls_color_themes()
```

## Value

vector of available color theme names, invisible

## Examples

``` r
ls_color_themes()
#> Available Data Dictionary color themes: 
#> Use 'show_color_theme()' to see the themes colors.
#> 
#> * aquamarin 
#> * dzhk 
#> * napkon 
#> * num3 
#> * no_colors 
#> * no_colours 
#> * purpur 
#> * my_theme 
```
