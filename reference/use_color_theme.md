# Set Color Theme That's Gonna Be Used For The Data Dictionary

Set Color Theme That's Gonna Be Used For The Data Dictionary

## Usage

``` r
use_color_theme(name)
```

## Arguments

- name:

  of color theme. Use [`ls_color_themes()`](ls_color_themes.md) to list
  all currently available themes.

## Value

invisibly the name parameter of this function call

## Examples

``` r
use_color_theme("purpur")
```
